### 🎓 საბაკალავრო პროექტი (კომპიუტერული მეცნიერება)

**ხელმძღვანელი:** ვლადიმერ სვანაძე


## პროექტის მიმოხილვა
პროექტის მიზანია შეიქმნას საგანმანათლებლო პლატფორმა, რომელიც ახდენს ფიშინგ-შეტევების სიმულაციას რეალურ გარემოში. სისტემა ორიენტირებულია არა მხოლოდ შეტევის ტექნიკურ დემონსტრირებაზე, არამედ მომხმარებელთა ცნობიერების ამაღლებაზე "Just-in-Time Learning" მეთოდოლოგიით.

## ძირითადი ფუნქციონალი
* **Phishing Templates:** რეალისტური Facebook, Instagram და Gmail ავტორიზაციის გვერდების სიმულაცია.
* **Admin Dashboard:** სტატისტიკური მონაცემების აღრიცხვა და ანალიზი.
* **Awareness Module:** მომხმარებლისთვის დაშვებული შეცდომების მომენტალური ახსნა და უსაფრთხოების რეკომენდაციები.
* **Security & Ethics:** მონაცემთა ანონიმურობა და GDPR რეგულაციების დაცვა.

## ტექნოლოგიური სტეკი
* **Backend:** Python 3.x, Flask Framework
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** SQLite3
* **Version Control:** Git & GitHub
