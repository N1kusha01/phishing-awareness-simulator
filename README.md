# ფიშინგ-სიმულატორი: სოციალური ინჟინერიის შეტევების მოდელირება

### 🎓 საბაკალავრო პროექტი (კომპიუტერული მეცნიერება)
**ავტორი:** ნიკა სეხნიაშვილი
**ხელმძღვანელი:** ვლადიმერ სვანაძე


## პროექტის მიმოხილვა
პროექტის მიზანია შეიქმნას საგანმანათლებლო პლატფორმა, რომელიც ახდენს ფიშინგ-შეტევების სიმულაციას რეალურ გარემოში. სისტემა ორიენტირებულია არა მხოლოდ შეტევის ტექნიკურ დემონსტრირებაზე, არამედ მომხმარებელთა ცნობიერების ამაღლებაზე "Just-in-Time Learning" მეთოდოლოგიით.

## ძირითადი ფუნქციონალი
* **ადმინისტრატორის მენიუ:** ცენტრალური პანელი, სადაც ხდება სასურველი პლატფორმის შერჩევა და სიმულაციური ბმულის გენერირება.
* **Phishing Templates:** რეალისტური Facebook, Instagram და Gmail ავტორიზაციის გვერდების სიმულაცია.
* **Data Logging:** მომხმარებლის მიერ შეყვანილი მონაცემების უსაფრთხო ლოგირება ლოკალურ ფაილში(log.txt)
* **Awareness Module:** მომხმარებლისთვის დაშვებული შეცდომების მომენტალური ახსნა და უსაფრთხოების რეკომენდაციები.
* **Security & Ethics:** მონაცემთა ანონიმურობა და GDPR რეგულაციების დაცვა.


## ტექნოლოგიური სტეკი
* **Backend:** Python 3.x, Flask Framework
* **Frontend:** HTML5, CSS3
* **Logging:** "log.txt" ფაილი
* **Dependencies:** მართვა ხდება "requirements.txt" ფაილის საშუალებით, რომელიც მოიცავს: 'blinker(1.9.0)', 'click(8.3.1)', 'colorama(0.4.6)', 'Flask(3.1.3)', 'itsdangerous(2.2.0)', 'Jinja2(3.1.6)', 'MarkupSafe(3.0.3)', 'Werkzeug(3.1.6)'.
* **Version Control:** Git & GitHub

  
## როგორ გავუშვათ პროქტი
1. ჩამოტვირთეთ პროექტის საქაღალდე.
2. დარწმუნდით, რომ კომპიუტერში დაინსტალირებულია **Python 3.x**.
3. ორჯერ დააკლიკეთ საქაღალდეში არსებულ start.bat(თუ Windows გაქვთ) ან start.command(თუ macOS გაქვთ) ფაილს.

* **რას აკეთებს სკრიპტი ავტომატურად:**
* ქმნის იზოლირებულ ვირტუალურ გარემოს(venv)
* 'requirements.txt' ფაილის მიხედვით ინტერნეტიდან იწერს ყველა საჭირო ბიბლიოთეკას.
* ავტომატურად ხსნის ბრაუზერს მისამართზე: 'http://127.0.0.1:5000'.
